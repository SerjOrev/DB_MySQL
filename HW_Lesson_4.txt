если хотом сделать лайки не только для messages а для любой сущности (users, media, friendships)
то, наверно, лучше сделать отдельную таблицу лайков для каждой сущности отдельно.
пример:
create table messages_likes (
id int,
message_id int,
user_id int,
like_type blob,
created_on date
)
